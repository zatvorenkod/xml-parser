# xml-parser
Simple xml parser written in C

# Credits/Resources used to learn

- [Intro to XML](https://www.stat.berkeley.edu/~nolan/stat133/Fall05/lectures/introXML.pdf).


# How to compile
- You need clang/gcc (test on macos)
```bash
chmod +x build.sh
build.sh
./xml_parser
```
